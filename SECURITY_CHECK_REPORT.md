## Рекомендации

### 🚀 Перед первым запуском

```bash
# 1. Скопировать шаблон окружения
cp .env.example .env

# 2. Отредактировать .env и добавить ключи
# Windows: notepad .env
# Linux/Mac: nano .env

# 3. Активировать виртуальное окружение
.\venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac

# 4. Запустить проект
cd assistant_giga  # или assistant_api
python app.py
```

### 🔐 Настройка ключей доступа

| Режим | Переменные | Где получить |
|-------|------------|--------------|
| GigaChat | `GIGACHAT_AUTH_KEY`, `GIGACHAT_RQUID` | [Сбер Cloud Platform](https://sbercloud.ru) |
| OpenAI API | `OPENAI_API_KEY` | [OpenAI Platform](https://platform.openai.com) |
| ProxiAPI | `PROXI_API_URL`, `PROXI_API_KEY` | Провайдер ProxiAPI |

### 📌 Важные заметки

1. **Никогда не коммитьте `.env`** - даже с пустыми значениями
2. **Регулярно обновляйте зависимости** - `pip install --upgrade -r requirements.txt`
3. **Используйте разные ключи** для development и production
4. **Резервируйте .env.example** - обновляйте при добавлении новых переменных
